2. Lancement de l'application
*******************************

Partie technique :

- Charger la sauvegarde de la base de donnée
- Utiliser le fichier TypesTable.sql pour créer les tables dans votre base de données
- Renseigner le chemin de connexion dans les settings de l'application via le fichier de config fourni
	
	
Partie utilisateur :

- Utiliser le .exe pour lancer l'application
- Référez vous a la doc client pour les détails d'utilisation
